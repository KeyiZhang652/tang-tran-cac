# 藏珍阁 · Tàng Trân Các — 网站使用说明

这是为您岳父打造的收藏展示网站。**双击 `index.html` 即可在浏览器中打开**，无需安装任何软件。

## 一、语言切换

页面右上角有 **VI / 中文 / EN / FR** 四个按钮（越南语、中文、英语、法语）。

- 首次打开时会自动根据访客浏览器的语言显示（默认越南语）；
- 访客手动切换后，下次打开会记住上次的选择。

## 二、如何放入真实照片（最重要）

把照片文件放进 `photos/` 文件夹，并按以下文件名命名，页面会**自动显示**（照片缺失时显示优雅的占位图，不影响页面美观）：

| 文件名 | 显示位置 |
|---|---|
| `photos/portrait.jpg` | "关于藏主"版块的肖像照 |
| `photos/01.jpg` | 珍品掠影 第 1 张（默认说明：ODO 摆钟） |
| `photos/02.jpg` | 第 2 张（Honda 67 摩托车） |
| `photos/03.jpg` | 第 3 张（Peugeot 自行车） |
| `photos/04.jpg` | 第 4 张（瑞士怀表） |
| `photos/05.jpg` | 第 5 张（钵场青釉水缸） |
| `photos/06.jpg` | 第 6 张（红木罗汉床） |

> 建议：照片裁成竖版（宽高比约 4:5）效果最佳；每张控制在 500KB 以内加载更快。

**修改照片说明文字**：用文本编辑器打开 `index.html`，搜索 `"g1"`、`"g2"`……即可找到每张照片在四种语言里的说明文字，直接改引号里的内容即可。

## 四、如何发布到网上（让别人访问）

任选其一，都免费：

- **最简单：Netlify Drop** — 打开 <https://app.netlify.com/drop>，把整个 `网页` 文件夹拖进去，几秒钟就有一个网址；
- **GitHub Pages** — 建一个仓库，把文件传上去，在设置里开启 Pages；
- 也可以先不发布，直接把整个文件夹拷给亲友，双击打开即可。

## 五、其他说明

- 网站名"藏珍阁 / Tàng Trân Các"意为"珍藏之阁"，中越通用（越南语为汉越音）。若想改名，搜索 `TÀNG TRÂN CÁC` 和 `藏珍阁` 替换；
- 字体通过 Google Fonts 在线加载，断网时会自动退回系统字体，页面依然可看；
- 六大门类的介绍文字里特意提到了越南收藏圈的经典元素（ODO 摆钟、Honda 67、Peugeot 自行车、钵场/边和水缸、红酸枝家具），懂行的藏友一看就知道是同道中人。
