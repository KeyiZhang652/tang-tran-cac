# 藏珍阁 · Tàng Trân Các — 网站使用说明

这是为您岳父打造的收藏展示网站。**双击 `index.html` 即可在浏览器中打开**，无需安装任何软件。

## 一、语言切换

页面右上角有 **VI / 中文 / EN / FR** 四个按钮（越南语、中文、英语、法语）。

- 首次打开时会自动根据访客浏览器的语言显示（默认越南语）；
- 访客手动切换后，下次打开会记住上次的选择。

## 二、六大门类图集（每张卡片里的横滑照片，最重要）

"一宅六藏"的每张类别卡片里都有一条**可以左右滑动的照片集**（手机上手指滑动，电脑上悬停出现箭头），点任意照片可**全屏查看**，全屏时左右滑动切换、点 ✕ 或点背景关闭。

**照片按文件夹自动读取，你只管往文件夹里放照片：**

| 文件夹 | 对应门类 |
|---|---|
| `photos/clocks/` | 古董钟表 |
| `photos/pendulum/` | 老式摆钟 |
| `photos/moto/` | 老式摩托车 |
| `photos/bike/` | 老式自行车 |
| `photos/jar/` | 老水缸 |
| `photos/wood/` | 红木家具 |

**命名规则（务必遵守）**：每个文件夹里按 `1.jpg`、`2.jpg`、`3.jpg`……**从 1 开始连续编号**，最多 12 张；也支持 `.png`、`.webp` 格式。网页会从 1 号开始依次找，**编号断了就停**（比如有 1、2、4 号，则只显示前两张）。替换照片＝用同名文件覆盖；增删照片＝保持编号连续。改完刷新网页即生效，不用改任何代码。

> 当前每个文件夹里的 6 张照片是我从维基共享资源（Wikimedia Commons）下载的**授权干净的示意图**（授权明细见 `PHOTO-CREDITS.md`），建议尽快替换成岳父的真实藏品照片——横版、宽 1200px 左右、每张 500KB 以内最佳。

> 建议：照片裁成竖版（宽高比约 4:5）效果最佳；每张控制在 500KB 以内加载更快。

**全屏大图展位（苹果官网式滚动效果，强烈建议上传）**：

| 文件名 | 显示位置 |
|---|---|
| `photos/feature-1.jpg` | 第一章全屏展位（ODO 摆钟） |

> 这三张是整个网站视觉冲击力最强的位置：滚动时图片会缓缓放大定格、文字浮现。请选**横版、高清**的照片（宽度 ≥1600px，宽高比约 16:10，每张 1MB 以内），拍摄时物件居中、背景干净最出效果。没上传时会显示雕版线稿占位图，不影响浏览。

**修改照片说明文字**：用文本编辑器打开 `index.html`，搜索 `"g1"`、`"g2"`……即可找到每张照片在四种语言里的说明文字，直接改引号里的内容即可。

### 以后如何更新网站内容

改完本地文件后，在终端进入 `网页` 文件夹执行：

```bash
git add -A
git commit -m "更新内容"
git push
```

推送后约 1 分钟，网站自动更新。

**不想用命令行？** 也可以直接在浏览器里操作：打开仓库页面 → 进入 `photos` 文件夹 → 点 "Add file → Upload files" 把照片拖进去提交即可；改文字则点开 `index.html` 后按铅笔图标在线编辑。

## 六、背景音乐

- 曲目：**肖邦《G大调夜曲》Op. 37 No. 2**（Olga Gurevich 演奏，来自 Musopen，CC0 公有领域授权，可放心商用/公开使用，无需署名）；
- 文件位置：`music/nocturne.mp3`，想换曲子就用同名文件替换（找音乐时务必确认是 CC0/公有领域授权）；
- 音量默认 22%（轻柔背景）。想调整就在 `index.html` 里搜索 `MUSIC_VOL`，改成 0～1 之间的数；
- 右下角的**喇叭图标**随时可以关闭/打开音乐，访客的选择会被记住（关过一次，下次访问就不会自动响）；
- 注意：所有浏览器都禁止网页自动出声，所以音乐会在访客**第一次点击或触摸页面后**才轻轻响起，这是正常现象，不是故障。
